# ДБН В.2.5-67:2013 - Опалення, вентиляція та кондиціонування

**Status:** Діючий
**Дата набрання чинності:** 2013-10-01
**ЄДЕССБ Registration:** Verify on e-construction.gov.ua
**Заміняє:** СНіП 2.04.05-91\*

## Сфера застосування

Встановлює вимоги до систем:

- Опалення
- Вентиляції
- Кондиціонування повітря
- Теплових мереж

Для будівель різного призначення в Україні.

## Ключові розділи

1. Загальні положення
2. Теплотехнічні вимоги до огороджувальних конструкцій
3. Опалення
4. Вентиляція та кондиціонування
5. Теплові мережі
6. Теплопостачання та теплоспоживання
7. Енергоефективність

## Ключові терміни

- **Теплова мережа** - комплекс трубопроводів для транспортування теплоносія від джерела до споживачів
- **Теплопункт** - комплекс обладнання для передачі теплової енергії від зовнішніх мереж до внутрішніх систем
- **Тепловий лічильник** - прилад для обліку кількості теплової енергії
- **Регулювання теплопостачання** - зміна параметрів теплоносія залежно від навантаження
- **Автоматизація теплопункту** - автоматичне регулювання параметрів теплоносія

## KRTM Usage

**Том 2 (Основні рішення):**

- Розділ "Теплопостачання" - технічні характеристики теплових мереж

**Технічна специфікація (ТЗ):**

- Параметри теплоносія (температура, тиск)
- Вимоги до теплових лічильників (метрологія)
- Вимоги до автоматизації котелень

**65 котелень:**

- Кожна котельня є джерелом теплопостачання для локальної мережі
- Обладнання котелень (котли, насоси, теплообмінники)
- Системи автоматизації та регулювання

**Система диспетчеризації KRTM:**

- Моніторинг параметрів теплоносія (температура подачі/зворотки, тиск, витрата)
- Контроль роботи обладнання котелень
- Енергоефективність - розрахунок KPI/SEC (див. розділ 3.3 ТЗ)

**Теплові лічильники (КВП - комплекси вимірювання параметрів):**

- ДБН В.2.5-67:2013 встановлює вимоги до метрологічної точності
- Інтеграція з системою ThingsBoard через Modbus RTU/TCP
