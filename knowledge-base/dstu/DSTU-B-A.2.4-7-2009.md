# ДСТУ Б А.2.4-7:2009 - Основні вимоги до проектної та робочої документації

**Status:** Діючий
**Дата набрання чинності:** 2010-09-01
**ЄДЕССБ Registration:** Verify on e-construction.gov.ua
**Заміняє:** ГОСТ 21.101-97 (частково)

## Сфера застосування

Встановлює:

- Загальні вимоги до виконання текстових і графічних документів
- Основні написи (title blocks / stamps) для креслень
- Оформлення пояснювальних записок
- Формати аркушів
- Шрифти та написи

## Ключові розділи

1. Загальні положення
2. Формати
3. Основні написи (title blocks)
4. Графічне оформлення
5. Текстові документи
6. Специфікації обладнання
7. Відомості матеріалів

## Основні написи (Title Blocks)

**Форма 1** - для креслень:

- Найменування об'єкта будівництва
- Найменування креслення
- Шифр проекту
- Стадія проектування (П, РП, Р)
- Аркуш, листів
- Організація-проектувальник
- Підписи (розробив, перевірив, затвердив)
- Дата

**Форма 2** - для текстових документів:

- Найменування організації
- Найменування документу
- Шифр проекту
- Том, частина
- Кількість аркушів
- Підписи

**Форма 3** - для специфікацій обладнання:

- Позначення
- Найменування
- Кількість
- Маса одиниці
- Примітка

## Ключові терміни

- **Основний напис** - штамп на кресленні або документі з реквізитами
- **Шифр проекту** - унікальний код проекту (наприклад, KRTM-2025-001)
- **Стадія проектування** - П (проект), РП (робочий проект), Р (робоча документація)
- **Формат аркуша** - розміри (А4, А3, А2, А1, А0)
- **Масштаб** - співвідношення розмірів на кресленні до натуральних розмірів

## KRTM Usage

**Всі томи (1-5):**

- Основні написи на титульних сторінках
- Оформлення пояснювальних записок
- Формати аркушів для документів

**Креслення (Том 3, Том 5):**

- Основні написи креслень (Форма 1)
- Шифр проекту KRTM
- Підписи відповідальних осіб

**Специфікації обладнання (Том 3):**

- Відомості обладнання ЦДП (сервери, комутатори, UPS)
- Відомості обладнання Edge Nodes (промислові ПК, модулі IoT Gateway)
- Специфікації за Формою 3

**Templates для KRTM:**

- Title page template (титульна сторінка томів)
- Drawing stamp template (штамп креслень)
- Equipment specification template (специфікація обладнання)
