# ГОСТ 2.104-2006 (ЄСКД) - Основні написи

**Status:** Діючий (прийнятий в Україні)
**Система:** ЄСКД (Єдина система конструкторської документації)
**Дата набрання чинності:** 2006-03-01
**Застосування:** Reference для креслень (в Україні - ДСТУ Б А.2.4-7 має пріоритет)

## Сфера застосування

Встановлює:

- Форми та розміри основних написів (штампів) на конструкторських документах
- Порядок заповнення граф основних написів
- Специфікації складальних одиниць

## Основні написи

**Форма 1** - для креслень (185 × 55 мм):

```
┌────────────────────────────────────────────────────────┐
│  Найменування виробу/деталі                            │
│  Позначення документа            Літера  Маса  Масштаб│
│  Розробив     Підпис   Дата      Аркуш   Аркушів       │
│  Перевірив    Підпис   Дата                            │
│  Т.контр.     Підпис   Дата      Організація           │
│  Затвердив    Підпис   Дата                            │
└────────────────────────────────────────────────────────┘
```

**Форма 2** - для наступних аркушів креслень (185 × 14 мм):

```
┌────────────────────────────────────────────────────────┐
│  Позначення документа            Літера  Аркуш         │
└────────────────────────────────────────────────────────┘
```

**Форма 2а** - для специфікацій (185 × 40 мм)

## Ключові поля

- **Позначення документа** - унікальний шифр креслення
- **Літера** - стадія проектування (П - проект, Р - робоча документація)
- **Маса** - маса виробу в кг
- **Масштаб** - 1:1, 1:2, 1:5, 1:10, 2:1 тощо
- **Аркуш/Аркушів** - нумерація та загальна кількість
- **Розробив/Перевірив/Затвердив** - підписи відповідальних осіб

## KRTM Usage

**Креслення обладнання (Том 3, Том 5):**

- Монтажні креслення Шаф ТМ (Телеметрії) для 65 котелень
- Схеми підключення Edge Nodes (Modbus RS-485 топологія)
- Кресленнs серверної ЦДП (розміщення обладнання, стійки)
- Схеми кабельних з'єднань

**Електротехнічні креслення:**

- Однолінійні схеми живлення обладнання
- Схеми заземлення та блискавкозахисту
- Кабельні журнали

**Специфікації:**

- Відомості обладнання за Формою 2а
- Кабельні вироби
- Комутаційне обладнання

## Coordination with DSTU

В Україні діє **ДСТУ Б А.2.4-7:2009**, який адаптує ГОСТ 2.104 для будівництва:

- Розміри штампів можуть відрізнятися
- Поля адаптовані під українську практику
- **ДСТУ має пріоритет для KRTM проекту**

Але ГОСТ 2.104 корисний як **reference** для:

- Машинобудівного обладнання (котли, насоси)
- Електротехнічних шаф (виробники часто використовують ГОСТ)
- Імпортного обладнання з документацією в ЄСКД
