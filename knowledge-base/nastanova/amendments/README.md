### Зміна №4 (20.03.2024) - Інфляційні витрати та інженер-консультант

**Документ:** [Zmina-4-2024.md](Zmina-4-2024.md)

**Ключові зміни:**

#### 1. **Пункт 5.33 (НОВИЙ)** - Інфляційні витрати

**Повна заміна методики визначення інфляційних витрат.**

**Формула:**

```
Інфляційні витрати = ПВ × Тб × ((Іінф/100) / 12)

де:
ПВ = Прямі витрати (ПВм + ПВе + ПВмаш + ПВінші)
Тб = Тривалість будівництва (місяці)
Іінф = Річний індекс інфляції (%)
```

**Приклад для KRTM:**

```
BOM (з Зміною №2): 21,000,000 UAH
Тривалість: 6 місяців
Індекс інфляції: 8% (2024)

Інфляційні витрати = 21,000,000 × 6 × (8/100 / 12)
                    = 21,000,000 × 6 × 0.00667
                    = 840,420 UAH
```

**KRTM Impact:**

- Додаткові витрати: **~840,000 UAH** (4% від BOM)
- Обов'язкове включення в Tom 4

#### 2. **Пункт 4.32.5 (оновлений)** - Оплата інженера-консультанта

**Зміни:**

- Тепер дозволяє **фіксовану оплату** за етапами
- Раніше: тільки % від вартості будівництва

**Методи оплати:**

- a) % від кошторисної вартості: **0.5-2.5%** залежно від категорії складності
- b) **Фіксована оплата** за етапами (Додаток 47 - НОВИЙ)

**Приклад фіксованої оплати для KRTM (Категорія ІІІ):**

| Етап      | Роботи                       | Вартість (UAH)  |
| --------- | ---------------------------- | --------------- |
| 1         | Аналіз ПКД + План якості     | 80,000          |
| 2         | Супровід будівництва (6 міс) | 360,000         |
| 3         | Приймання ПНР                | 60,000          |
| **Разом** |                              | **500,000 UAH** |

**KRTM Recommendation:**

- Використовувати **фіксовану оплату** (краща передбачуваність)
- Додати до Tom 4 як окремий розділ

#### 3. **Додаток 47 (НОВИЙ)** - Форма акту для фіксованої оплати

**Назва:** "Акт прийняття послуг інженера-консультанта (фіксована оплата за етапами)"

**Структура:**

- Реквізити сторін (Замовник, Інженер-консультант)
- Опис етапу робіт
- Фіксована вартість етапу
- Перелік виконаних робіт
- Підписи та печатки

**KRTM Application:**

- Використовувати для всіх 3 етапів (аналіз ПКД, супровід, приймання)
- Інтегрувати в Tom 1, Section 9 (Договір)

#### 4. **Інші зміни:**

- Пункт 4.41 (абзац 2): коригування регіональних коефіцієнтів
- Додатки 43, 44, 45: оновлення форм актів

**KRTM Impact:**

- **Інфляційні витрати:** +840,000 UAH (4%)
- **Інженер-консультант:** +500,000 UAH (фіксована оплата)
- **Загальне подорожчання:** ~1,340,000 UAH

**Статус:** ✅ Повний текст з офіційного PDF
