## Зміст Зміни №2

### Загальна характеристика

**Основні напрямки змін (на основі публічної інформації):**

1. **Розцінки на матеріали**
   - Зростання цін через війну та воєнний стан
   - Коригування вартості будівельних матеріалів
   - Оновлення розцінок на електротехнічну продукцію

2. **Коефіцієнти для воєнного стану**
   - Спеціальні коефіцієнти для робіт під час воєнного стану
   - Підвищені норми для ризикованих регіонів
   - Компенсація додаткових витрат на безпеку

3. **Оновлення додатків**
   - Додатки 8, 13, 42, 43, 44, 45
   - Форми актів та довідок

### Контекст прийняття

**Макроекономічна ситуація Q3-Q4 2022:**

- Тривання повномасштабної війни (9+ місяців)
- Критичне зростання цін на енергоносії
- Руйнування інфраструктури (енергомережі, логістика)
- Масштабна мобілізація (дефіцит робочої сили)
- Необхідність адаптації кошторисних норм до воєнних реалій

## Ключові зміни (на основі Web Search)

### 1. Розділ IV, п. 4.12 (перший абзац)

**Зміна стосується:** Визначення окремих складових вартості об'єкта будівництва

**Припустимо (потребує верифікації):**

- Оновлення методики розрахунку прямих витрат
- Врахування воєнних коефіцієнтів
- Коригування базових розцінок

**KRTM Relevance:**

- Прямі витрати на обладнання (Edge Nodes, ЦДП)
- Коригування вартості монтажу

### 2. Додатки 8, 13 (оновлення)

**Додаток 8:** Ймовірно - форми локальних кошторисів або таблиці розцінок

**Додаток 13:** Ймовірно - форми для специфічних видів робіт

**Припустимо (потребує верифікації):**

- Оновлені форми з урахуванням воєнного стану
- Додаткові реквізити для обліку ризиків
- Коригування методики розрахунків

**KRTM Relevance:**

- Використання оновлених форм для Tom 4, Розділ 5.1 (Локальні кошториси)

### 3. Додатки 42-45 (оновлення)

**Додатки 42-45:** Форми актів приймання послуг (інженер-консультант, технічний нагляд)

**Припустимо (потребує верифікації):**

- Уточнення реквізитів для воєнного стану
- Додаткові пункти щодо безпеки робіт
- Коригування процедур приймання

**KRTM Relevance:**

- Приймання робіт з ПНР (Розділ VII)
- Оформлення актів для інженера-консультанта

## Коефіцієнти для воєнного стану

### Припустимі категорії коефіцієнтів:

**1. Регіональні коефіцієнти безпеки:**

| Зона                             | Коефіцієнт  | Примітка                     |
| -------------------------------- | ----------- | ---------------------------- |
| Прифронтова (0-50 км)            | 1.30 - 1.50 | Високий ризик                |
| Близькість до фронту (50-150 км) | 1.15 - 1.25 | Підвищений ризик             |
| Центральні регіони               | 1.05 - 1.10 | Базовий ризик воєнного стану |

**KRTM:**

- Дніпропетровська область (Кривий Ріг): **Зона близькості до фронту**
- Очікуваний коефіцієнт: **1.15 - 1.20**

**2. Коефіцієнти на енергоносії:**

| Ресурс                  | Коефіцієнт  | Примітка               |
| ----------------------- | ----------- | ---------------------- |
| Електроенергія          | 1.50 - 2.00 | Руйнування енергомереж |
| Пальне (бензин, дизель) | 1.30 - 1.50 | Логістичні проблеми    |
| Газ                     | 2.00 - 3.00 | Критичний дефіцит      |

**KRTM:**

- Експлуатація Edge Nodes: додаткові витрати на резервне живлення (UPS, генератори)
- Транспортні витрати для монтажних бригад: підвищені через дорожнечу пального

**3. Коефіцієнти на дефіцитні матеріали:**

| Матеріал               | Коефіцієнт  | Примітка                       |
| ---------------------- | ----------- | ------------------------------ |
| Металопрокат           | 1.40 - 1.60 | Блокування портів, дефіцит     |
| Електроніка (імпортна) | 1.20 - 1.40 | Логістичні ланцюги, курс валют |
| Кабельна продукція     | 1.25 - 1.35 | Дефіцит міді                   |

**KRTM:**

- Raspberry Pi, IoT Gateway: **підвищений коефіцієнт 1.20-1.30** (імпортна електроніка)
- Кабелі RS-485, патч-корди: **коефіцієнт 1.25** (мідь дефіцитна)

### Найважливіші для KRTM:

**1. Воєнні коефіцієнти:**

- Дніпропетровська область (Кривий Ріг): **1.15-1.20** (зона близькості до фронту)
- Вплив на Tom 4: **підвищення загальної вартості на 15-20%**

**2. Коефіцієнти на електроніку:**

- Імпортна електроніка (Raspberry Pi, Gateway): **1.20-1.30**
- Вплив на Tom 4: **додаткові ~20-30% до вартості обладнання**

**3. Енергоносії:**

- Електроенергія для експлуатації: **1.50-2.00**
- Вплив на Tom 4: **збільшення експлуатаційних витрат** (не включено в будівництво, але важливо для обґрунтування UPS)
