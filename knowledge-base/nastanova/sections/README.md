### Розділ II. Система ціноутворення у будівництві

**Файл:** [`section-2-pricing-system.md`](./section-2-pricing-system.md)

**Зміст:**

- Принципи ціноутворення (база, види цін)
- Детальний опис методів визначення вартості:
  - **Ресурсний** (найточніший, рекомендований для KRTM)
  - **Ресурсно-індексний** (з застосуванням індексів КМУ)
  - **Аналоговий** (на основі аналогічних об'єктів)
  - **Параметричний** (питомі показники: грн/м², грн/Edge Node)
- Формування складових вартості (ПВ, ЗВВ, КП, додаткові, непередбачені)
- Регіональні коефіцієнти (Дніпропетровська область згідно Зміни №4)

**KRTM Relevance:** CRITICAL - визначає методику розрахунку

**Ключові формули:**

```
Вартість_БМР = ПВ + ЗВВ + КП
ЗВВ = ФОП × Норматив_ЗВВ (%)
КП = ФОП × Норматив_КП (%)
Загальна вартість = ПВ + ЗВВ + КП + Додаткові + Непередбачені + ПДВ
```

### Розділ IV. Визначення окремих складових вартості об'єкта будівництва

**Файл:** [`section-4-cost-components.md`](./section-4-cost-components.md)

**Зміст:**

- **Кошторисний прибуток (КП):**
  - Норми: 50-75% (залежить від виду робіт)
  - **KRTM:** Монтаж Edge - 60%, ПНР - 70%
- **Загальновиробничі витрати (ЗВВ):**
  - Склад ЗВВ (адмін-господарські, обслуговування працівників, організація робіт)
  - Норми: 75-140% (залежить від виду робіт)
  - **KRTM:** Монтаж Edge - 94%, ПНР - 125%
- **Додаткові витрати:**
  - Здешевлення в зимовий період
  - Тимчасові будівлі та споруди (1-3%)
  - **KRTM:** 1.5% (тимчасові склади обладнання)
- **Непередбачені витрати:**
  - Норми: 2-3% (нове будівництво), 3-5% (реконструкція)
  - **KRTM:** 3% (інноваційний IoT проєкт)
- **Інфляційні витрати (Зміна №4, п. 5.33):**
  - Методика розрахунку (прогнозні індекси КМУ або Мінекономіки)
  - **KRTM:** Рекомендовано врахувати (проєкт 6-12 місяців)
- **Технічний нагляд та інженер-консультант (Зміна №4, п. 4.32.5):**
  - Форма акта (Додаток 47 - NEW)

**KRTM Relevance:** CRITICAL - всі норми для розрахунку вартості

**Зведена таблиця нормативів KRTM:**
| Складова | Норматив | База |
|----------|----------|------|
| ЗВВ (монтаж Edge) | 94% | від ФОП |
| КП (монтаж Edge) | 60% | від ФОП |
| ЗВВ (ПНР) | 125% | від ФОП |
| КП (ПНР) | 70% | від ФОП |
| Тимчасові | 1.5% | від (ПВ+ЗВВ+КП) |
| Непередбачені | 3.0% | від (ПВ+ЗВВ+КП+Тимчасові) |

### Розділ V. Визначення вартості об'єкта будівництва при складанні ціни пропозиції учасника процедури закупівлі

**Файл:** [`section-5-procurement-pricing.md`](./section-5-procurement-pricing.md)

**Зміст:**

- Очікувана вартість закупівлі (замовник розраховує до тендеру)
- Ціна пропозиції учасника (підрядник складає для тендеру)
- Договірна ціна (результат переговорів/тендеру)
- Види договірних цін (тверда, відкрита, з відшкодуванням витрат)
- **Інфляційні витрати (Зміна №4, п. 5.33 - повний перевиклад):**
  - Методика розрахунку
  - Прогнозні індекси (КМУ або Мінекономіки під час воєнного стану)
  - Включення в договір (clause з індексацією)
- Оцінка пропозицій учасників (критерії: ціна, якість, кваліфікація)
- Аномально низька ціна (< 80% від очікуваної)

**KRTM Relevance:** HIGH - тендерні процедури, договірна ціна

**Рекомендації KRTM:**

- Вид договірної ціни: **відкрита** з індексацією (проєкт 6-12 місяців)
- Критерій оцінки: **якість + ціна** (балова система 100 балів: ціна 40, технічні характеристики 25, досвід 20, гарантія 10, строки 5)
- Процедура закупівлі: Відкриті торги (ЗУ "Про публічні закупівлі", ст. 23)

### Розділ VI. Визначення вартості виконаних робіт та витрат

**Файл:** [`section-6-completed-works.md`](./section-6-completed-works.md)

**Зміст:**

- Визначення обсягів виконаних робіт (фізичні виміри, за проектними обсягами)
- Виконавча документація (as-built drawings, протоколи випробувань, паспорти)
- **Акти приймання виконаних робіт:**
  - Форма КБ-2в (будівельні роботи)
  - **Форма КБ-3** (монтажні роботи) - основна для KRTM
  - Довідка КБ-3в (для бухгалтерії)
- Акт приймання змонтованого обладнання
- **Акт приймання виконаних ПНР:**
  - Перелік виконаних робіт
  - Протоколи випробувань (функціональні, продуктивність, відмовостійкість)
  - Критерії успішності (безперервна робота 24-48 год, автоматичне відновлення)
- Розрахунки з підрядником:
  - Авансові платежі (10-30%, KRTM: 20%)
  - Проміжні платежі (щомісячно, утримання авансу та гарантійних коштів)
  - Остаточний розрахунок
  - Гарантійні кошти (5-10%, утримуються на термін гарантії)

**KRTM Relevance:** HIGH - контроль виконання, платежі підряднику

**Графік платежів KRTM:**

- Аванс: 4,600,000 грн (20%)
- Щомісячні платежі: згідно КБ-3 мінус утримання
- Гарантійні кошти: 1,150,000 грн (5%) - повернення після 24 міс

### Розділ VII. Особливості визначення вартості пусконалагоджувальних робіт

**Файл:** [`section-7-commissioning.md`](./section-7-commissioning.md)

**Зміст:**

- Поняття ПНР (пусконалагоджувальні роботи)
- Відмінності ПНР від монтажу (результат - працююча система, не фізичні одиниці)
- **Склад пусконалагоджувальних робіт:**
  - Етап 1: Підготовчі роботи
  - Етап 2: Автономні випробування
  - Етап 3: Комплексні випробування
  - Етап 4: Приймальні випробування
  - Етап 5: Дослідна експлуатація (Pilot Operation, 30+ днів)
- **Визначення вартості ПНР:**
  - Ресурсний метод (рекомендований)
  - Норми ЗВВ: 120-140% (KRTM: 125%)
  - Норми КП: 65-75% (KRTM: 70%)
  - Обґрунтування високих норм (висока кваліфікація, складність, інтелектуальна праця)
- **KRTM: Специфіка ПНР для IoT системи:**
  - Edge Node (типова котельня): 4 дні інженер-налагоджувач, 38,740 грн
  - ЦДП: 20 днів (команда інженерів), 360,800 грн
  - Дослідна експлуатація: 30 днів, 270,000 грн
  - **ВСЬОГО ПНР KRTM:** 2,897,990 грн (~12-15% від загальної вартості проєкту)
- Приймання результатів ПНР (критерії успішності, акт приймання)
- Документація ПНР (протоколи, as-built configs, інструкції)

**KRTM Relevance:** **CRITICAL** - ПНР є ключовою фазою впровадження IoT системи

**⚠️ ВАЖЛИВО:** ПНР для IoT систем становить 12-15% від загальної вартості (типове співвідношення для складних систем автоматизації). Недооцінка ПНР - поширена помилка.
