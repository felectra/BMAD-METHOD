## 2.1. Загальні принципи ціноутворення

### 2.1.1. База ціноутворення

Вартість будівництва визначається на основі:

1. **Прямих витрат (ПВ)** на виконання будівельно-монтажних робіт
2. **Загальновиробничих витрат (ЗВВ)** на організацію будівництва
3. **Кошторисного прибутку (КП)** будівельної організації
4. **Додаткових витрат** на тимчасові споруди, здешевлення та ін.
5. **Непередбачених витрат** (резерв на ризики)

### 2.1.2. Види цін

**Базисна вартість** - вартість у цінах базисного року (історичні ціни):

- Використовується для порівняння
- Коригується індексами до поточних цін

**Поточна вартість** - вартість у цінах на дату складання кошторису:

- Основа для фінансування
- Відображає актуальну ринкову ситуацію

**Договірна ціна** - ціна, узгоджена в договорі підряду:

- Може відрізнятися від кошторисної вартості
- Результат переговорів та конкурентних процедур

## 2.2. Методи визначення вартості будівництва

### 2.2.1. Ресурсний метод

**Принцип:** Вартість визначається як сума вартості всіх ресурсів за поточними цінами.

**Формула:**

```
ПВ = Σ (Qмат × Цмат) + Σ (Qпраці × Тариф) + Σ (Qмаш × Вартість_маш-год)
```

де:

- **Qмат** - кількість матеріалів, виробів, конструкцій
- **Цмат** - поточна ціна матеріалів
- **Qпраці** - трудомісткість (людино-години)
- **Тариф** - тарифна ставка оплати праці
- **Qмаш** - машино-години експлуатації техніки
- **Вартість_маш-год** - вартість 1 машино-години

**Переваги:**

- Найбільш точний метод
- Відображає реальну ринкову ситуацію
- Рекомендований для нового будівництва

**Недоліки:**

- Трудомісткий
- Потребує детальних специфікацій

**Застосування в KRTM:**

- ✅ Ідеальний для IoT обладнання (маємо детальний BOM)
- ✅ Consolidated BOM: 18,342,726 грн з ПДВ
- Додати: монтажні роботи, ПНР, ЗВВ, КП

### 2.2.2. Ресурсно-індексний метод

**Принцип:** Базисна вартість (в історичних цінах) множиться на індекси для отримання поточної вартості.

**Формула:**

```
Поточна вартість = Базисна вартість × Індекс
```

**Індекси:**

- **Індекси цін виробників промислової продукції** (Держстат)
- **Галузеві індекси** (будівництво, монтаж, обладнання)
- **Регіональні коефіцієнти** (наприклад, для Дніпропетровської області)

**Переваги:**

- Швидкість розрахунку
- Простота застосування

**Недоліки:**

- Менша точність
- Залежність від актуальності індексів

**Застосування в KRTM:**

- Можливо для типових монтажних робіт
- Індекси КМУ для Дніпропетровської області (Зміна №4)

### 2.2.3. Аналоговий метод

**Принцип:** Вартість визначається на основі вартості аналогічних об'єктів з коригуванням.

**Формула:**

```
Вартість_проєкту = Вартість_аналога × Кпод × Кскл × Крег × Кінф
```

де:

- **Кпод** - коефіцієнт подібності (відмінності в параметрах)
- **Кскл** - коефіцієнт складності
- **Крег** - регіональний коефіцієнт
- **Кінф** - коефіцієнт інфляції (якщо аналог старий)

**Переваги:**

- Швидкість на ранніх стадіях проекту
- Не потребує детальних специфікацій

**Недоліки:**

- Низька точність
- Складно знайти адекватний аналог

**Застосування в KRTM:**

- ❌ Не рекомендовано (немає аналогів для IoT системи такого масштабу)

### 2.2.4. Параметричний метод

**Принцип:** Вартість визначається через питомі показники.

**Формула:**

```
Вартість = Питомий показник × Параметр об'єкта
```

**Приклади питомих показників:**

- Будівлі: грн/м² (площа), грн/м³ (об'єм)
- Дороги: грн/км
- Мережі: грн/км, грн/точку підключення
- **IoT системи: грн/Edge Node, грн/датчик** ← потенційно для KRTM

**Переваги:**

- Швидкість розрахунку
- Зручність для планування

**Недоліки:**

- Низька точність
- Не враховує специфіку конкретного проекту

**Застосування в KRTM:**

- Можливо на стадії попередньої оцінки:
  - **Вартість однієї котельні (Edge Node)** × 65
  - Економія масштабу при закупівлі обладнання

## 2.3. Формування складових вартості

### 2.3.1. Прямі витрати (ПВ)

**Склад:**

1. **Вартість матеріалів, виробів, конструкцій:**
   - Ціна постачальника
   - Транспортні витрати до будівельного майданчика
   - Заготівельно-складські витрати (ЗСВ): 1-3%

2. **Оплата праці робітників:**
   - Основна заробітна плата
   - Додаткова заробітна плата (відпустки, премії)
   - ЄСВ (єдиний соціальний внесок): 22% від ФОП

3. **Експлуатація будівельних машин і механізмів:**
   - Вартість машино-години
   - Оплата праці машиністів

**Формула:**

```
ПВ = Матеріали + ФОП_робітників + ЄСВ + Машини
```

**KRTM приклад:**

```
ПВ (обладнання) = 18,342,726 грн (BOM) + ЗСВ (1-3%) + Транспорт
ПВ (монтаж) = ФОП_монтажників + ЄСВ (22%) + Спецтехніка
```

### 2.3.2. Загальновиробничі витрати (ЗВВ)

**Раніше:** Накладні витрати (НВ)

**Склад:**

- Адміністративно-господарські витрати будівельної організації
- Заробітна плата ІТР, службовців, МОП
- Утримання офісу, транспорту
- Охорона праці, техніка безпеки

**Норматив:** % від ПВ (без вартості матеріалів) або від ФОП

**Формула:**

```
ЗВВ = (ПВ - Матеріали) × Норматив_ЗВВ (%)
```

**Типові норми ЗВВ:**

- Будівельні роботи: 90-110%
- Монтажні роботи: 75-95%
- Пусконалагоджувальні роботи: 120-140%

**KRTM:**

- Монтаж Edge Node обладнання: ~80-90%
- ПНР IoT системи: ~120-130%

### 2.3.3. Кошторисний прибуток (КП)

**Призначення:** Планові накопичення будівельної організації для розвитку та модернізації.

**Норматив:** % від ПВ (без вартості матеріалів) або від ФОП

**Формула:**

```
КП = (ПВ - Матеріали) × Норматив_КП (%)
```

**Типові норми КП:**

- Будівельні роботи: 60-70%
- Монтажні роботи: 50-65%
- ПНР: 65-75%

### 2.3.4. Кошторисна вартість будівельно-монтажних робіт

**Формула:**

```
Вартість_БМР = ПВ + ЗВВ + КП
```

### 2.3.5. Додаткові витрати

1. **Здешевлення будівництва в зимовий період** (якщо застосовно)
2. **Тимчасові будівлі та споруди:** 1-3% від Вартості_БМР
3. **Інші витрати** (відповідно до проекту)

### 2.3.6. Непередбачені витрати

**Призначення:** Резерв на покриття ризиків та непередбачених обставин.

**Норматив:**

- Нове будівництво: **2-3%** від суми (ПВ + ЗВВ + КП + Додаткові витрати)
- Реконструкція: **3-5%**
- Капітальний ремонт: до 10%

**Формула:**

```
Непередбачені = (Вартість_БМР + Додаткові витрати) × 0.02-0.03
```

**KRTM:** 2-3% (нове будівництво IoT системи)

### 2.3.7. Загальна кошторисна вартість

**Формула:**

```
Загальна вартість = ПВ + ЗВВ + КП + Додаткові + Непередбачені + ПДВ
```

де **ПДВ** = 20% (якщо не включено в ціни ресурсів)

## 2.4. Регіональні та інші коефіцієнти

### 2.4.1. Регіональні коефіцієнти

**Призначення:** Коригування вартості з урахуванням регіональних особливостей (клімат, транспортна доступність, місцеві умови).

**Дніпропетровська область:** Див. Зміна №4 (2024) - актуальні коефіцієнти

**Застосування:**

```
Вартість_регіональна = Вартість_базова × К_регіональний
```

### 2.4.2. Індекси інфляції

**Джерело:** Кабінет Міністрів України (КМУ постанови)

**Типи індексів:**

- Індекси цін виробників промислової продукції
- Індекси споживчих цін
- Галузеві індекси (будівництво, монтаж)

**Оновлення:** Щоквартально (Зміна №1-4 інтегрують актуальні індекси)

### Приклад розрахунку для KRTM

**1. Прямі витрати (обладнання):**

```
ПВ_обладнання = 18,342,726 грн (з ПДВ, BOM Tom 3)
```

**2. Прямі витрати (монтаж):**

```
ПВ_монтаж = ФОП_монтажників + ЄСВ (22%) + Транспорт + Спецтехніка
```

**3. ЗВВ (монтаж):**

```
ЗВВ = ПВ_монтаж × 0.85 (85% для електромонтажних робіт)
```

**4. Кошторисний прибуток:**

```
КП = ПВ_монтаж × 0.60 (60%)
```

**5. ПНР (пусконалагоджувальні роботи):**

```
ПВ_ПНР = ФОП_інженерів + ЄСВ (22%) + Витратні матеріали
ЗВВ_ПНР = ПВ_ПНР × 1.25 (125% для ПНР)
КП_ПНР = ПВ_ПНР × 0.70 (70%)
```

**6. Непередбачені:**

```
Непередбачені = (ПВ + ЗВВ + КП) × 0.03 (3%)
```

**7. Загальна вартість:**

```
КОШТОРИСНА ВАРТІСТЬ = Сума всіх складових
```
