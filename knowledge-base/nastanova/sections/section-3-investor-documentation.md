## 3.1. Локальні кошториси

### 3.1.1. Призначення

**Локальний кошторис** - кошторисний документ, що визначає вартість окремого виду будівельних, монтажних або інших робіт на основі обсягів робіт, передбачених проектною документацією.

**Складається на:**

- Окремі види будівельних робіт (фундаменти, стіни, покрівля)
- Окремі види монтажних робіт (електромонтаж, сантехніка, вентиляція)
- Окремі види обладнання (придбання та монтаж)
- Пусконалагоджувальні роботи (ПНР)

### 3.1.2. Структура локального кошторису

**Основні розділи:**

1. **Шапка (метадані):**
   - Найменування об'єкта будівництва
   - Найменування робіт
   - Підстава (проектна документація, креслення)
   - Дата складання

2. **Таблична частина:**
   | № | Найменування робіт | Од. виміру | Кількість | Ціна за од. | Загальна вартість |
   |---|-------------------|-----------|-----------|------------|-------------------|
   | | | | | | |

3. **Прямі витрати (ПВ):**
   - Матеріали, вироби, конструкції
   - Оплата праці робітників
   - Експлуатація машин і механізмів

4. **Загальновиробничі витрати (ЗВВ):**
   - % від (ПВ - Матеріали) або від ФОП

5. **Кошторисний прибуток (КП):**
   - % від (ПВ - Матеріали) або від ФОП

6. **Разом по локальному кошторису:**
   - Підсумок: ПВ + ЗВВ + КП

### 3.1.3. Методика розрахунку

**Покрокова інструкція:**

**Крок 1: Визначення обсягів робіт**

- Взяти обсяги з проектної документації
- Відповідність одиницям виміру (м², м³, м, шт, т, кВт тощо)

**Крок 2: Визначення ПВ (ресурсний метод)**

```
ПВ = Σ (Обсяг_i × Ціна_ресурсу_i)
```

де:

- **Обсяг_i** - кількість матеріалів, людино-годин, машино-годин
- **Ціна_ресурсу_i** - поточна ціна на дату складання кошторису

**Крок 3: Розрахунок ЗВВ**

```
ЗВВ = (ПВ - Матеріали) × Норматив_ЗВВ (%)
```

або

```
ЗВВ = ФОП_робітників × Норматив_ЗВВ (%)
```

**Крок 4: Розрахунок КП**

```
КП = (ПВ - Матеріали) × Норматив_КП (%)
```

**Крок 5: Підсумок**

```
Вартість_локальна = ПВ + ЗВВ + КП
```

### 3.1.4. Форми локальних кошторисів

**Додатки до Настанови:**

- **Додаток 1** - Локальний кошторис (ресурсний метод)
- **Додаток 2** - Локальний кошторис (ресурсно-індексний метод)
- **Додаток 3-15** - Специфічні форми (обладнання, ПНР, демонтаж тощо)

## 3.2. Об'єктні кошториси

### 3.2.1. Призначення

**Об'єктний кошторис** - зведення локальних кошторисів за будівлею, спорудою або об'єктом в цілому.

**Складається на:**

- Окремі будівлі (наприклад, адмінбудівля)
- Окремі споруди (наприклад, ЦДП - серверна)
- Групу об'єктів (наприклад, всі Edge Nodes)

### 3.2.2. Структура об'єктного кошторису

**Табличний формат:**

| №   | Найменування локального кошторису  | Номер локального кошторису | Вартість, грн |
| --- | ---------------------------------- | -------------------------- | ------------- |
| 1   | Локальний кошторис №1              | ЛК-01                      | XXX           |
| 2   | Локальний кошторис №2              | ЛК-02                      | YYY           |
| ... | ...                                | ...                        | ...           |
|     | **Разом по об'єктному кошторису:** |                            | **ΣXYZ**      |

**Додатково:**

- Тимчасові будівлі та споруди (1-3% від суми локальних)
- Інші витрати (якщо застосовно)
- Непередбачені витрати (2-3%)

### 3.2.3. Методика складання

**Крок 1:** Згрупувати локальні кошториси за об'єктами

**Крок 2:** Підсумувати вартість локальних кошторисів

**Крок 3:** Додати додаткові витрати:

```
Додаткові витрати = Σ(Локальні) × 0.01-0.03
```

**Крок 4:** Додати непередбачені витрати:

```
Непередбачені = (Σ(Локальні) + Додаткові) × 0.02-0.03
```

**Крок 5:** Підсумок:

```
Вартість_об'єктна = Σ(Локальні) + Додаткові + Непередбачені
```

### 3.2.4. Форми об'єктних кошторисів

**Додатки до Настанови:**

- **Додаток 16** - Об'єктний кошторис (основна форма)
- **Додаток 17-25** - Специфічні форми (реконструкція, капремонт тощо)

## 3.3. Зведений кошторисний розрахунок

### 3.3.1. Призначення

**Зведений кошторисний розрахунок (ЗКР)** - кошторисний документ, що визначає **повну вартість будівництва** всього проєкту, включаючи всі об'єкти, роботи та витрати.

**Складається на:**

- Весь проєкт будівництва в цілому
- Всі об'єкти (будівлі, споруди, мережі)
- Всі витрати (проектування, будівництво, обладнання, резерви)

### 3.3.2. Структура ЗКР (Глава 1-12)

**Типова структура згідно Настанови:**

| Глава  | Найменування                           | Склад витрат                                             |
| ------ | -------------------------------------- | -------------------------------------------------------- |
| **1**  | Підготовка території будівництва       | Знесення, вертикальне планування, винос комунікацій      |
| **2**  | Основні об'єкти будівництва            | Будівлі, споруди (за об'єктними кошторисами)             |
| **3**  | Об'єкти допоміжного призначення        | Трансформаторні підстанції, котельні, водонапірні вежі   |
| **4**  | Об'єкти енергетичного господарства     | Електромережі, теплопостачання, водопровід, каналізація  |
| **5**  | Об'єкти транспортного господарства     | Дороги, під'їзні шляхи, стоянки                          |
| **6**  | Зовнішні мережі та споруди             | Зовнішні інженерні мережі                                |
| **7**  | Благоустрій та озеленення              | Озеленення, малі архітектурні форми                      |
| **8**  | Тимчасові будівлі та споруди           | 1-3% від Глав 1-7                                        |
| **9**  | Інші роботи та витрати                 | Авторський нагляд, технічний нагляд, інженер-консультант |
| **10** | Проектні та вишукувальні роботи        | Вартість проектування (якщо не окремо)                   |
| **11** | Резерв коштів на непередбачені витрати | 2-3% від Глав 1-10                                       |
| **12** | Податки, збори, обов'язкові платежі    | ПДВ 20% (якщо застосовно)                                |
|        | **ЗАГАЛЬНА КОШТОРИСНА ВАРТІСТЬ**       | **ΣГЛАВ 1-12**                                           |

### 3.3.3. Методика складання ЗКР

**Покрокова інструкція:**

**Крок 1: Розподіл об'єктних кошторисів по главах**

- Кожен об'єктний кошторис → відповідна Глава ЗКР

**Крок 2: Підсумок основних витрат (Глави 1-7)**

```
Основні витрати = Σ(Глави 1-7)
```

**Крок 3: Тимчасові будівлі (Глава 8)**

```
Глава 8 = Основні витрати × 0.01-0.03
```

**Крок 4: Інші роботи (Глава 9)**

- Авторський нагляд: 0.5-1% від вартості будівництва
- Технічний нагляд: 1-2%
- Інженер-консультант: за договором

**Крок 5: Проектні роботи (Глава 10)**

- Якщо не оплачені окремо: 3-7% від вартості будівництва
- **KRTM:** 786,000 грн (меморандум ЮНІСЕФ) - вже оплачено окремо

**Крок 6: Непередбачені (Глава 11)**

```
Глава 11 = Σ(Глави 1-10) × 0.02-0.03
```

**Крок 7: ПДВ (Глава 12)**

- Якщо ПДВ не включено в ціни ресурсів:

```
Глава 12 = Σ(Глави 1-11) × 0.20
```

**Крок 8: Загальна кошторисна вартість**

```
ЗАГАЛЬНА ВАРТІСТЬ = Σ(Глави 1-12)
```

## 3.4. Особливості для різних видів будівництва

### 3.4.1. Нове будівництво

- Повний комплект кошторисів (локальні → об'єктні → ЗКР)
- Непередбачені: **2-3%**
- Всі 12 глав ЗКР (де застосовно)

**KRTM:** Нове будівництво IoT системи (проте встановлення в існуючі котельні)

### 3.4.2. Реконструкція

- Врахування демонтажу старого обладнання
- Непередбачені: **3-5%** (більше через невизначеність стану існуючих конструкцій)
- Окремі локальні кошториси на демонтаж

### 3.4.3. Капітальний ремонт

- Локальні кошториси на ремонтні роботи
- Непередбачені: **до 10%** (високий рівень невизначеності)
- Часто обмежений об'єм робіт

### 3.4.4. Технічне переоснащення

- Заміна обладнання без зміни конструкцій
- Локальні кошториси: демонтаж + монтаж нового обладнання
- Непередбачені: **2-4%**
