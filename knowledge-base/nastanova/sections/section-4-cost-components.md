# Розділ IV. Визначення окремих складових вартості об'єкта будівництва

**Джерело:** НАСТАНОВА з визначення вартості будівництва (Наказ №281 від 01.11.2021)
**Розділ:** IV
**Статус:** Діючий з 2021-11-01
**Останні зміни:** **Зміна №4 (20.03.2024)** ← Важливі правки в п. 4.32.5 та 4.41

## 4.1. Загальні положення

Цей розділ встановлює правила визначення окремих складових вартості будівництва:

- **Загальновиробничі витрати (ЗВВ)**
- **Кошторисний прибуток (КП)**
- **Додаткові витрати**
- **Непередбачені витрати**
- **Витрати на технічний нагляд та інженера-консультанта**

---

## 4.2. Кошторисний прибуток (КП)

### 4.2.1. Призначення

**Кошторисний прибуток** - це планові накопичення будівельної (підрядної) організації, призначені для:

- Розвитку виробництва
- Оновлення основних фондів
- Стимулювання працівників
- Соціальний розвиток колективу
- Сплата податків

### 4.2.2. Методика визначення

**Базова формула:**

```
КП = (ПВ - Матеріали - Обладнання) × Норматив_КП (%)
```

або (спрощено):

```
КП = ФОП_робітників × Норматив_КП (%)
```

де:

- **ПВ** - прямі витрати
- **Матеріали** - вартість матеріалів, виробів, конструкцій
- **Обладнання** - вартість обладнання (якщо окремо виділено)
- **ФОП_робітників** - фонд оплати праці робітників

### 4.2.3. Нормативи КП

**Типові норми кошторисного прибутку:**

| Вид робіт                             | Норматив КП, % | База нарахування     |
| ------------------------------------- | -------------- | -------------------- |
| **Будівельні роботи**                 | 60-70%         | від (ПВ - Матеріали) |
| **Монтажні роботи (загальні)**        | 50-65%         | від (ПВ - Матеріали) |
| **Монтаж санітарно-технічних систем** | 55-65%         | від ФОП              |
| **Монтаж електротехнічних установок** | 50-60%         | від ФОП              |
| **Монтаж слабкострумових систем**     | 60-70%         | від ФОП              |
| **Пусконалагоджувальні роботи (ПНР)** | 65-75%         | від ФОП              |
| **Ремонтні роботи**                   | 55-65%         | від ФОП              |

**KRTM Application:**

- Монтаж Edge Node обладнання (електротехніка + слабкострумові): **60%** від ФОП
- ПНР IoT системи: **70%** від ФОП

### 4.2.4. Особливості визначення

**Коли КП НЕ нараховується:**

- На вартість обладнання, що не потребує монтажу
- На вартість матеріалів замовника (давальницька схема)
- На витрати, що відшкодовуються окремо

---

## 4.3. Загальновиробничі витрати (ЗВВ)

### 4.3.1. Призначення

**ЗВВ** (раніше - "накладні витрати") - це витрати будівельної організації на:

- Організацію і управління виробництвом
- Обслуговування будівельного процесу
- Створення необхідних умов для виконання робіт

### 4.3.2. Склад ЗВВ

**Групи витрат:**

1. **Адміністративно-господарські витрати:**
   - Заробітна плата адміністративного персоналу (директор, бухгалтер, кадровик)
   - Заробітна плата ІТР (інженери, майстри, прораби)
   - Утримання офісу (оренда, комунальні, зв'язок)
   - Канцелярські витрати
   - Послуги банків

2. **Витрати на обслуговування працівників:**
   - Охорона праці та техніка безпеки
   - Медичне обслуговування
   - Спецодяг та засоби індивідуального захисту
   - Забезпечення санітарно-побутових умов

3. **Витрати на організацію робіт:**
   - Геодезичні роботи
   - Лабораторні випробування
   - Утримання будівельних майданчиків
   - Утримання і експлуатація виробничих фондів

4. **Витрати на підвищення кваліфікації:**
   - Навчання та перекваліфікація кадрів
   - Сертифікація працівників

5. **Інші витрати:**
   - Страхування працівників та майна
   - Податки та збори (крім ПДВ та податку на прибуток)

### 4.3.3. Методика визначення

**Базова формула:**

```
ЗВВ = (ПВ - Матеріали - Обладнання) × Норматив_ЗВВ (%)
```

або (спрощено):

```
ЗВВ = ФОП_робітників × Норматив_ЗВВ (%)
```

### 4.3.4. Нормативи ЗВВ

**Типові норми загальновиробничих витрат:**

| Вид робіт                             | Норматив ЗВВ, % | База нарахування     |
| ------------------------------------- | --------------- | -------------------- |
| **Будівельні роботи (загальні)**      | 90-110%         | від (ПВ - Матеріали) |
| **Монтажні роботи (загальні)**        | 75-95%          | від (ПВ - Матеріали) |
| **Монтаж санітарно-технічних систем** | 80-90%          | від ФОП              |
| **Монтаж електротехнічних установок** | 75-85%          | від ФОП              |
| **Монтаж слабкострумових систем**     | 85-95%          | від ФОП              |
| **Пусконалагоджувальні роботи (ПНР)** | 120-140%        | від ФОП              |
| **Ремонтні роботи**                   | 95-115%         | від ФОП              |
| **Монтаж технологічного обладнання**  | 70-80%          | від ФОП              |

**Підвищувальні коефіцієнти:**

- Віддалені об'єкти (за межами міста): +10-15%
- Роботи у важкодоступних місцях: +15-25%
- Роботи в зимовий період: +5-10%

**KRTM Application:**

- **Монтаж Edge Node обладнання:** 85% від ФОП
  - Електротехніка + слабкострумові системи
  - 65 віддалених об'єктів (котельні по всьому місту): +10%
  - Підсумок: **85% × 1.10 = 93.5%** ≈ **94% від ФОП**

- **ПНР IoT системи:** 125% від ФОП
  - Складне налаштування ThingsBoard, Modbus, MQTT
  - Інтеграція 65 Edge Nodes з ЦДП

- **Монтаж ЦДП (серверна):** 80% від ФОП
  - Технологічне обладнання (сервери, мережа)
  - Стаціонарний об'єкт

### 4.3.5. Особливості визначення

**Коли ЗВВ НЕ нараховуються:**

- На вартість обладнання, що не потребує монтажу
- На вартість матеріалів замовника
- На роботи, виконувані господарським способом

## 4.4. Додаткові витрати

### 4.4.1. Здешевлення будівництва в зимовий період

**Призначення:** Компенсація додаткових витрат на:

- Обігрів будівельних майданчиків
- Протиморозні добавки в розчини
- Снігоочищення
- Додаткові трудовитрати через знижену продуктивність

**Методика визначення:**

```
Здешевлення_зимове = Вартість_робіт_у_зимовий_період × Коеф_здешевлення
```

**Коефіцієнти здешевлення:**

- Залежать від регіону (кліматичні зони України)
- Дніпропетровська область: див. регіональні нормативи

**KRTM:** Потенційно застосовно, якщо монтаж плануєтьсяв зимовий період (грудень-лютий)

### 4.4.2. Тимчасові будівлі та споруди

**Призначення:** Витрати на:

- Тимчасові офіси на будмайданчику
- Склади матеріалів та обладнання
- Санітарно-побутові приміщення для робітників
- Огорожі будівельних майданчиків
- Тимчасові мережі (електро, вода)

**Методика визначення:**

```
Тимчасові = Вартість_БМР × 0.01-0.03
```

де **Вартість_БМР** = ПВ + ЗВВ + КП (без додаткових та непередбачених)

**Типові норми:**

- Нове будівництво (великий об'єкт): 2-3%
- Реконструкція: 1.5-2.5%
- Монтаж обладнання (без будівельних робіт): 1-1.5%

**KRTM:** **1.5%** (монтаж обладнання в існуючі котельні, тимчасові склади для обладнання)

### 4.4.3. Інші додаткові витрати

**Можуть включати:**

- Витрати на експлуатацію будівель та споруд після завершення будівництва до прийняття в експлуатацію
- Витрати на утримання дирекції будівництва
- Інші витрати, передбачені проектом

**KRTM:** Не застосовно

## 4.5. Непередбачені витрати

### 4.5.1. Призначення

**Непередбачені витрати** - резерв коштів на покриття витрат, які неможливо передбачити на стадії проектування:

- Зміни в проектних рішеннях під час будівництва
- Додаткові роботи, виявлені в процесі будівництва
- Підвищення цін на ресурси (якщо не враховано окремо)
- Інші непередбачені обставини

### 4.5.2. Методика визначення

**Формула:**

```
Непередбачені = (ПВ + ЗВВ + КП + Додаткові витрати) × Норматив (%)
```

### 4.5.3. Нормативи непередбачених витрат

**Згідно Настанови:**

| Вид будівництва            | Норматив, % | База                            |
| -------------------------- | ----------- | ------------------------------- |
| **Нове будівництва**       | 2-3%        | від (ПВ + ЗВВ + КП + Додаткові) |
| **Реконструкція**          | 3-5%        | від (ПВ + ЗВВ + КП + Додаткові) |
| **Капітальний ремонт**     | 5-10%       | від (ПВ + ЗВВ + КП + Додаткові) |
| **Технічне переоснащення** | 2-4%        | від (ПВ + ЗВВ + КП + Додаткові) |

**Підвищувальні коефіцієнти:**

- Інноваційні проєкти (нові технології): +0.5-1%
- Складні умови (високий рівень невизначеності): +1-2%

**KRTM Application:**

- **Тип:** Нове будівництво IoT системи
- **Інноваційність:** Перший проєкт такого масштабу для КТМ
- **Рекомендований норматив:** **3%**

```
Непередбачені_KRTM = (ПВ + ЗВВ + КП + Тимчасові) × 0.03
```

### 4.5.4. Використання резерву непередбачених витрат

**Правила:**

- Витрачається тільки на непередбачені роботи та витрати
- Потребує обґрунтування та затвердження замовником
- Не використовується для покриття помилок проектування або неякісного виконання робіт
- Облік використання ведеться окремо

**KRTM:** Резерв має бути затверджений інвестором (ЮНІСЕФ), використання - за узгодженням

## 4.6. Інфляційні витрати (Зміна №4)

### 4.6.1. Кошти на покриття інфляційних процесів

**Зміна №4 (п. 5.33)** вводить можливість врахування коштів на покриття інфляції:

**Призначення:** Відшкодування збільшення вартості ресурсів внаслідок інфляції протягом будівництва.

**Застосування:**

- У ціні пропозиції учасника закупівлі
- У договірній ціні (якщо передбачено проектом)

**Методика розрахунку (згідно Зміни №4):**

```
Інфляційні кошти = f(Строки будівництва, Структура робіт, Вартість ресурсів, Прогнозні індекси КМУ)
```

**Вхідні дані:**

- **Строки будівництва** (місяці)
- **Структура робіт** (розподіл по періодах)
- **Вартість ресурсів** в базових цінах
- **Прогнозні індекси цін** виробників промислової продукції (КМУ постанови)

**Якщо індекси КМУ відсутні** (під час дії воєнного стану):

- Використовуються прогнозні індекси від центрального органу виконавчої влади (Мінекономіки)

**KRTM:** Якщо будівництво триватиме > 6 місяців, рекомендується врахувати інфляційні кошти (орієнтовно 5-10% річних)

## 4.7. Витрати на технічний нагляд та інженера-консультанта

### 4.7.1. Технічний нагляд (Зміна №1, п. 4.32.5)

**Призначення:** Контроль за якістю та відповідністю робіт проекту.

**Норматив:**

```
Технічний нагляд = Вартість будівництва × 1.0-2.0%
```

**KRTM:** **1.5%** від вартості будівництва (контроль монтажу 65 Edge Nodes + ЦДП)

### 4.7.2. Інженер-консультант (Зміна №4, п. 4.32.5)

**Зміна №4** детально регулює взаєморозрахунки за послуги інженера-консультанта.

**Два варіанти оплати:**

**Варіант 1: Пропорційна оплата**

- Оплата пропорційно вартості фактично виконаних робіт
- Форми актів: Додаток 43, 44, 45 Настанови

**Варіант 2: Фіксовані суми**

- Якщо інженер-консультант залучений не на весь строк будівництва
- Або для окремих функцій
- **Форма акта: Додаток 47 Настанови** (введений Зміною №4)

**KRTM Application:**

- Можливо залучення ThingsBoard експерта (інженер-консультант)
- Функції: консультації з налаштування кластера HA, Rule Engine, Dashboard
- Оплата: фіксовані суми за етапами (Додаток 47)

## 4.8. Авторський нагляд

### 4.8.1. Призначення

**Авторський нагляд** - контроль архітектора/проектувальника за відповідністю будівництва проектним рішенням.

**Норматив:**

```
Авторський нагляд = Вартість проектування × 0.10-0.15
```

або

```
Авторський нагляд = Вартість будівництва × 0.5-1.0%
```

**KRTM:**

- Вартість проектування: 786,000 грн (меморандум ЮНІСЕФ)
- Авторський нагляд: 786,000 × 0.12 = **94,320 грн**
