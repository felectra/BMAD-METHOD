## Сфера застосування

Цей стандарт встановлює вимоги та рекомендації щодо визначення:
- Класу наслідків (відповідальності) об'єктів будівництва
- Категорії складності об'єктів будівництва

Призначено для застосування:
- Замовниками та проектувальниками
- Розробниками будівельних норм та стандартів
- Експертними організаціями
- Центральним органом виконавчої влади у сфері державного архітектурно-будівельного контролю

---

## Ключові терміни

### Класи наслідків (відповідальності): СС1, СС2, СС3

- **СС1** - незначні наслідки (низька відповідальність)
- **СС2** - середні наслідки (середня відповідальність)
- **СС3** - значні наслідки (висока відповідальність)

### Категорії складності: І, ІІ, ІІІ, ІV, V

Визначаються на підставі класу наслідків та характеристик об'єкту.

### Інші терміни

- **Відмова об'єкту** - стан об'єкту, при якому неможливо використовувати його за функціональним призначенням
- **Постійне перебування людей** - перебування більше 8 годин на добу та не менше 150 днів на рік (мінімум 1200 годин/рік)
- **Періодичне перебування людей** - перебування від 450 до 1200 годин на рік (не більше 8 годин на добу протягом максимум 150 днів)
- **Небезпека для життєдіяльності людей зовні об'єкту** - можливе порушення умов життєдіяльності більше ніж на три доби

|---|---|---|---|---|---|---|

## Методика визначення класу наслідків

### Крок 1: Визначити характеристики можливих наслідків

3. **Можлива небезпека для життєдіяльності людей, які перебувають зовні об'єкту (N₃)**
4. **Обсяг можливого економічного збитку (Ф)**
5. **Втрата об'єктів культурної спадщини**
6. **Припинення функціонування об'єктів інженерно-транспортної інфраструктури**

### Крок 2: Класифікувати кожну характеристику

### Крок 3: Вибрати найвищий клас

## Формули розрахунку

### Формула 5.1 - Кількість людей зовні об'єкту

```
N₃ = α × N₁
```

Де:
- N₃ = кількість людей зовні об'єкту
- α = коефіцієнт (таблиця 2)
- N₁ = кількість людей, які постійно перебувають на об'єкті

### Формула 5.2 - Прогнозовані економічні збитки

```
Ф = с × Вс × m
```

Де:
- Ф = прогнозовані збитки (тис. грн.)
- с = коефіцієнт втрат основних фондів (рекомендовано 0,45, але визначається на основі сценарію аварії)
- Вс = вартість основних фондів (тис. грн.)
- m = коефіцієнт залишкової вартості на момент можливої аварії

Збитки розраховуються на основі аварійного сценарію із врахуванням заходів локалізації.

## Методика розрахунку для житлового будинку (Приклад)

### Крок 1: Розрахунок N₁ (постійно присутні люди)

Для 24-поверхового 144-квартирного будинку:
- Визначити типологію квартир (1-кімнатні, 2-кімнатні, 3-кімнатні)
- Розрахувати кількість осіб за нормою (21 м² на особу + 10,5 м² на сім'ю)
- Позначити в таблиці розселення за кількістю кімнат
- Сумарна кількість осіб = N₁

**Приклад:** N₁ = 291 особа → **СС2** (від 50 до 400)

### Крок 2: Розрахунок N₂ (періодично присутні люди)

Для житлових будинків: N₂ = 50% від N₁

**Приклад:** N₂ = 145 осіб → **СС2** (від 100 до 1000)

### Крок 3: Розрахунок N₃ (люди зовні об'єкту)

```
N₃ = α × N₁ = 1,5 × 291 = 437 осіб
```

**Приклад:** N₃ = 437 осіб → **СС2** (від 100 до 50000)

### Крок 4: Розрахунок економічних збитків (Ф)

- Визначити площу будинку: S = 7608 м²
- Встановити розрахункову вартість за 1 м²: 5986 грн./м²
- Розрахувати вартість: Вс = 5986 × 7608 = 45541 тис. грн.
- Розрахувати збитки: Ф = 0,45 × 45541 = 20493 тис. грн. (рекомендовано 0,225, що дає 10247 тис. грн.)
- Конвертувати в м.р.з.п.: 10247 ÷ 1,102 = 9299 м.р.з.п.

**Результат:** 9299 м.р.з.п. → **СС2** (від 2000 до 150000)

### Крок 5: Розрахунок культурної спадщини

- Будинок НЕ розташований у охоронній зоні
- Будинок НЕ є об'єктом культурної спадщини
- **Результат:** немає наслідків (не враховується)

### Крок 6: Розрахунок впливу на транспортну інфраструктуру

- Відмова будинку не впливає на припинення роботи транспорту, зв'язку, енергетики
- **Результат:** немає наслідків (не враховується)

### Висновок за прикладом

| Параметр | Клас | Значення |
|---|---|---|
| N₁ (постійно) | СС2 | 291 осіб |
| N₂ (періодично) | СС2 | 145 осіб |
| N₃ (зовні) | СС2 | 437 осіб |
| Ф (економічні збитки) | СС2 | 9299 м.р.з.п. |
| Культурна спадщина | - | - |
| Транспортна інфраструктура | - | - |

**Найвищий клас:** СС2 → **Категорія складності: ІІІ**

## KRTM Usage - Розрахунок CC1, Category ІІ

### Обґрунтування вибору СС1 для KRTM

**KRTM проєкт (Комплексна Розподілена Телеінформаційна Мережа) - це система спеціалізованої інфраструктури, яка повинна бути класифікована як СС1 з наступних причин:**

#### 1. Людський фактор (низька щільність постійного персоналу)

- **Постійно присутні люди (N₁):** Максимум 20-30 операторів ЦДП (Центр даних та керування)
  - Це істотно менше за 50 осіб → **СС1**
- **Періодично присутні люди (N₂):** Техніки та інженери для обслуговування
  - Обслуговування передбачається 1-2 рази на місяць → **СС1**
- **Люди зовні об'єкту (N₃):** Edge Nodes розташовані у промислових зонах
  - Мінімальний вплив на населення поза об'єктом → **СС1**

#### 2. Економічні збитки (розумна вартість)

- **Вартість системи:** ~2-5 млн. грн. (типовий комплект ЦДП + 10 Edge Nodes)
- **Прогнозовані збитки при аварії:** ~500-1000 тис. грн.
- **У м.р.з.п.:** 500 ÷ 1,102 = ~450 м.р.з.п. → **СС1** (менше 2000)

#### 3. Культурна спадщина

- KRTM розташована у промислових зонах, не у охоронних зонах
- **Результат:** немає впливу → **СС1**

#### 4. Транспортна інфраструктура

- KRTM підтримує локальні комунікації та IoT, але не є критичною для загальнодержавної транспортної мережі
- Альтернативні маршрути комунікації існують → **СС1**

**ВИСНОВОК:** KRTM класифікується як **СС1, Категорія ІІ** на підставі таблиці А.1.

### Посилання на ТЗ розділ 1.2

Розділ 1.2 технічного завдання KRTM встановлює:
- Архітектура розподіленої системи з ЦДП та Edge Nodes
- Низька критичність персоналу (автоматизована система керування)
- Вартість компонентів у межах допустимого економічного збитку для СС1
- Відсутність критичної залежності від об'єкту для загальнодержавної інфраструктури

Таким чином, проєкт KRTM повністю відповідає критеріям СС1 і категорії ІІ складності.

## Спеціальні положення

### Об'єкти підвищеної небезпеки

Незалежно від інших характеристик, наступні об'єкти класифікуються як **СС3**:

- Об'єкти, ідентифіковані згідно із Законом України "Про об'єкти підвищеної техногенної небезпеки"
- Сховища цивільного захисту (цивільної оборони) незалежно від місцезнаходження та класу

### Сейсмічне проектування

Коефіцієнт α у Додатку Г для проектування у сейсмічних районах:

- **СС3 об'єкти:** γI ≤ 0,4
- **СС2 об'єкти:** γI ≤ 0,6
- **СС1 об'єкти:** γI ≤ 0,8

Визначається розрахункова сейсмічна інтенсивність для майданчика відповідно до ДБН В.1.1-12:2006.

## Примітки та уточнення

1. **Актуальність м.р.з.п.:** При розрахунку збитків використовувати поточний мінімальний розмір заробітної плати з Закону про Державний бюджет України на рік проектування.

2. **Максимальна вартість:** Рекомендується на етапі проектування розрахувати можливі варіанти збитків (мінімум, типовий, максимум) залежно від сценарію аварії.

3. **Коефіцієнт с:** Стандарт рекомендує с = 0,45, але допускає уточнення на підставі аналізу конкретного аварійного сценарію.

4. **Перевірка експертами:** Розраховану категорію складності перевіряють державні будівельні експерти при проведенні державної експертизи проекту.

5. **Уточнення на етапі Р:** Якщо на етапі робочої документації (Р) визначена категорія змінилася, до завдання на проектування вносяться корективи.
